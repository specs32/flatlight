The Library can be zipped and imported in Arduino(has to be greater version than 1.0.0) or can be directly copied in the
\Documents\Arduino\libraries\ProLib
directory to update the library you have to delete the old one manually and copy the new files in said directory.

The Library is still WIP and with the given .ino you can controll the LED Matrix.
Press the Moon Button to turn it on/off. Double press it to change the pattern. Hold it for Battery status.
